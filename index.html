Latest Projects

Implementing an IoT cloud based solution for monitoring servers room

The solution is to install a surveillance camera and temperature, humidity and air quality sensors in the room. The camera is part of a facial recognition module used to secure access to the room. The sensors will be used to extract the ambient state of the room in real time. This hardware will be connected to development boards connected to a AWS, which processes its data, will allow the servers room administrator to access real-time visualization dashboards and to be notified by email and SMS if a problem occurs: an unexpected rise in temperature for example.
